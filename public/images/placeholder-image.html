BBD Garage - Placeholder Görselleri

Aşağıdaki görsellerin oluşturulması gerekiyor:

  • hero-bg.jpg - Ana sayfa hero görseli
  • about-hero.jpg - Hakkımızda sayfası hero görseli
  • services-hero.jpg - Hizmetler sayfası hero görseli
  • store-hero.jpg - Mağaza sayfası hero görseli
  • contact-hero.jpg - İletişim sayfası hero görseli
  • about-image.jpg - Hakkımızda sayfası şirket görseli
  • team-member-1.jpg, team-member-2.jpg, team-member-3.jpg - Ekip üyeleri
  • product-1.jpg ila product-8.jpg - Mağaza ürün görselleri
İpucu: Bu görselleri hızlıca oluşturmak için Picsum Photos ya da Placeholder.com gibi placeholder görsel servisleri kullanabilirsiniz. Ya da kendi görsellerinizi ekleyebilirsiniz.

Örnek Kod:

Placeholder görselleri indirmek için PowerShell komutu:

$urls = @(
    "https://picsum.photos/seed/hero/1200/600 hero-bg.jpg",
    "https://picsum.photos/seed/about/1200/600 about-hero.jpg",
    "https://picsum.photos/seed/services/1200/600 services-hero.jpg",
    "https://picsum.photos/seed/store/1200/600 store-hero.jpg",
    "https://picsum.photos/seed/contact/1200/600 contact-hero.jpg",
    "https://picsum.photos/seed/company/600/400 about-image.jpg",
    "https://picsum.photos/seed/team1/300/300 team-member-1.jpg",
    "https://picsum.photos/seed/team2/300/300 team-member-2.jpg",
    "https://picsum.photos/seed/team3/300/300 team-member-3.jpg",
    "https://picsum.photos/seed/product1/400/300 product-1.jpg",
    "https://picsum.photos/seed/product2/400/300 product-2.jpg",
    "https://picsum.photos/seed/product3/400/300 product-3.jpg",
    "https://picsum.photos/seed/product4/400/300 product-4.jpg",
    "https://picsum.photos/seed/product5/400/300 product-5.jpg",
    "https://picsum.photos/seed/product6/400/300 product-6.jpg",
    "https://picsum.photos/seed/product7/400/300 product-7.jpg",
    "https://picsum.photos/seed/product8/400/300 product-8.jpg"
)

$outputPath = ".\public\images\"

foreach ($item in $urls) {
    $parts = $item -split " "
    $url = $parts[0]
    $filename = $parts[1]
    $outputFile = Join-Path $outputPath $filename
    
    Write-Host "İndiriliyor: $url -> $outputFile"
    Invoke-WebRequest -Uri $url -OutFile $outputFile
}

Write-Host "Tüm görseller indirildi!"
